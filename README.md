## Instructions

Explore the world by dragging Mr. Floppypants around:
* Pick up things
* Go for a swim
* Play golf
* Drive the car
* Totally wreck the dinner table

![Example Image](promo/dinner.png?raw=true)

## Running it locally

~~~
npm install
npm start
~~~
